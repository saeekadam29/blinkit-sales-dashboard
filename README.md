# 🛒 Blinkit Sales Performance Dashboard | Power BI

## 📌 Project Overview
An end-to-end Power BI dashboard analyzing $1.2M in sales across 8,500+ grocery 
items from Blinkit outlets. The dashboard uncovers outlet performance, product 
trends, and strategic growth opportunities across Tier 1, 2, and 3 locations.

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (M Language)
- Microsoft Excel (source data)

## 📊 Dashboard Pages
### Page 1 — Sales Overview
- KPI Cards: Total Sales ($1.2M), Avg Sales ($141), Avg Rating (3.92), No. of Items (9K)
- Historical Sales Trend (2010–2022) — 2018 peak at $205K
- Sales by Outlet Type — Supermarket Type 1 dominates at $787K
- Sales by Outlet Location Tier — Tier 3 drives highest revenue ($472K)
- Fat Content Analysis — Low Fat outsells Regular ($717K vs $409K)
- Top Item Categories — Fruits, Snacks, and Household lead sales
- Dynamic metric toggle — switch between Total Sales, Avg Sales, Rating, Items

### Page 2 — Insights & Strategy
- Key Performance Overview
- Critical business insights derived from data
- 3 Strategic Recommendations:
  1. Inventory Optimization — increase Low-Fat stock in Tier 3 outlets
  2. Service Training — replicate Type 1 best practices in Type 3 stores
  3. Bundle Strategy — create Fruit & Snack bundles for Tier 1 locations

## 💡 Key Findings
- Supermarket Type 1 is the clear market leader (65% of total sales)
- Tier 3 outlets generate highest revenue but show stagnant ratings — growth opportunity
- Low-Fat products consistently outperform Regular across all outlet tiers
- 2018 was peak sales year — post-2018 decline suggests need for market re-entry strategy
